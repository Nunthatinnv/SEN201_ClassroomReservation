# ScheDool - The Classroom Reserver
A simple, easy-to-use local program to reserve classrooms for subjects or competencies.

</div>

## Features
- Room Creation
- Reserve Classrooms
- View Reservations
- Export Reservations

## Quick Start / Installation
1. Download the ScheDool package from our [Releases Page](https://github.com/Nunthatinnv/SEN201_ClassroomReservation/releases/latest). Choose the appropriate package for your operating system, e.g. `schedool-linux-amd64.zip`
 for Linux. 
2. Once the zip file has been downloaded, extract the zip file, there would be a directory called `schedool`.
3. Enter the `schedool` directory, and run the `classroom_reservation` executable. 

**NOTE**: The executable can be ran via the terminal e.g `./classroom_reservation` or a single-click launch.
